# Requirements: Joel Shinness Website

**Defined:** 2026-02-09
**Core Value:** Small business owners can understand what Joel does, trust his process, and easily reach out to start a conversation.

## v1.2 Requirements

Requirements for Homepage Refinement milestone. Each maps to roadmap phases.

### Foundation

- [ ] **FOUND-01**: Icon library migrated from lucide-static to @lucide/astro for tree-shaking
- [ ] **FOUND-02**: Isometric CSS utilities added to global.css (transform-style: preserve-3d patterns)
- [ ] **FOUND-03**: Shadow utilities for isometric elements support dark mode glow transformation

### Hero

- [ ] **HERO-01**: Hero headline reframes from "what I build" to "outcomes you get" (time saved, money saved, customers satisfied)
- [ ] **HERO-02**: Visual outcome badges display up to 3 key metrics below headline
- [ ] **HERO-03**: Badges use design system shadow utilities and pass WCAG contrast checks

### Process

- [ ] **PROC-01**: Each of 5 process steps has 1-2 sentence description explaining what happens
- [ ] **PROC-02**: Each process step has an isometric mini-illustration
- [ ] **PROC-03**: Illustrations follow consistent style guide (45° light angle, neobrutalist palette, flat colors)
- [ ] **PROC-04**: Illustrations are optimized (<20KB each) and have descriptive alt text

### Technology

- [ ] **TECH-01**: Technology section reorganized into 3 categories: AI, Automations, Web Apps
- [ ] **TECH-02**: Each category has 1-2 sentence description explaining the offering
- [ ] **TECH-03**: Each category has an isometric illustration matching process section style
- [ ] **TECH-04**: Grid layout responsive (stacked mobile, 3-column desktop)

### FAQ

- [ ] **FAQ-01**: Dedicated /faq page created with existing FAQ content
- [ ] **FAQ-02**: FAQ page includes JSON-LD structured data for SEO
- [ ] **FAQ-03**: Navigation updated to include FAQ link
- [ ] **FAQ-04**: Footer FAQ section removed or replaced with link to /faq page
- [ ] **FAQ-05**: FAQ accordion passes keyboard accessibility (Enter/Space toggle, Tab navigation)

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multiple CTAs in hero | Reduces conversions by 266% per research |
| Animated isometric scenes | File size bloat, accessibility issues, static aesthetic is on-brand |
| Pricing calculator | Custom work varies too much, creates false expectations |
| Real-time chat widget | Overkill for freelance, contact form + FAQ covers communication |
