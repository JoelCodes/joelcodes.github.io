**Directory setup:**
Create `src/components/illustrations/` directory for SVG components.

**Style Guide (from research + established patterns):**
- **Projection:** 45-degree isometric angle (matches iso-rotate utility)
- **Shapes:** 3-5 simple geometric shapes per illustration
- **Colors:** Use `fill="currentColor"` - NO hard-coded hex colors
- **Opacity:** Use opacity (0.7, 0.85, 1.0) to create face differentiation
- **ViewBox:** Always include `viewBox="0 0 100 100"` for scaling
- **Size target:** Under 500 bytes unminified, under 300 bytes minified

**Illustration Concepts (simple geometric representations):**

1. **ProcessDiscovery.svg** - Magnifying glass or speech bubble
   - Concept: Discovery/understanding
   - Shapes: Circle + handle (magnifying glass) OR rounded rectangle + triangle (speech bubble)

2. **ProcessPrototype.svg** - Window/screen with content
   - Concept: Demo/preview
   - Shapes: Isometric rectangle (screen) with inner rectangles (content blocks)

3. **ProcessProposal.svg** - Document with checkmark
   - Concept: Agreement/scope
   - Shapes: Isometric rectangle (paper) + checkmark path + lines (text)

4. **ProcessBuild.svg** - Stacked blocks or gears
   - Concept: Construction/development
   - Shapes: 2-3 isometric cubes stacked OR interlocking gear shapes

5. **ProcessHandover.svg** - Gift box or rocket
   - Concept: Delivery/launch
   - Shapes: Isometric cube with bow OR simple rocket shape

**Template structure for each SVG:**
```svg
<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg" aria-hidden="true">
  <!-- Front face (opacity 0.85) -->
  <path d="..." fill="currentColor" opacity="0.85"/>
  <!-- Top face (opacity 1.0 - lightest) -->
  <path d="..." fill="currentColor" opacity="1"/>
  <!-- Side face (opacity 0.7 - darkest) -->
  <path d="..." fill="currentColor" opacity="0.7"/>
</svg>
```

**Important:**
- Include `aria-hidden="true"` since these are decorative (descriptive text is in the component)
- Keep paths minimal - use simple shapes, not detailed artwork
- Test that SVG renders clearly at 64x64px (desktop) and 48x48px (mobile)

**DO NOT use:**
- Hard-coded colors (#FFE600, rgb(), etc.)
- Complex gradients or filters
- Text elements within SVG
- Stroke-based designs (use fill only)
- More than 5 path elements per SVG
