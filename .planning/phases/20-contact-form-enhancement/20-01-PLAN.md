  <name>Task 1: Create CheckboxGroup component</name>
  <files>src/components/ui/CheckboxGroup.astro</files>
  <action>
Create a neobrutalist checkbox group component following the Input.astro pattern.

Props interface:
- name: string (required) - form field name for checkbox values
- legend: string (required) - accessible group label (renders in fieldset legend)
- options: Array<{value: string, label: string}> - checkbox options
- variant?: 'yellow' | 'turquoise' | 'magenta' (default: 'yellow')
- class?: string

Structure (from RESEARCH.md W3C WCAG H71 pattern):
```html
<fieldset class="checkbox-group">
  <legend class="checkbox-group-legend">
    {legend} <span class="optional-hint">(optional)</span>
  </legend>
  <div class="checkbox-items">
    {for each option:}
    <label class="checkbox-label">
      <input type="checkbox" name={name} value={option.value} class="checkbox-neo">
      <span class="checkbox-text">{option.label}</span>
    </label>
  </div>
</fieldset>
```

Neobrutalist checkbox styling (from RESEARCH.md neobrutalism.dev patterns):
- appearance: none (custom styling)
- Size: 1.25rem x 1.25rem
- Border: 3px solid var(--color-text-light)
- Background: var(--color-bg-light)
- border-radius: 0.25rem
- Checked state: background uses variant color, 3px 3px 0 shadow
- Focus state: double ring technique (2px bg gap + 4px text ring)
- Dark mode: invert bg/text, use variant-dark colors

Add subtle hover effect on entire label (slight scale or background tint) for larger interactive area.
