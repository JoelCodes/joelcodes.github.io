<task type="auto">
  <name>Task 1: Create dedicated FAQ page with JSON-LD schema</name>
  <files>src/pages/faq.astro</files>
  <action>
Create src/pages/faq.astro following established page patterns.

Structure:
1. Import BaseLayout from "../layouts/BaseLayout.astro"
2. Define FAQ data array with question/answer pairs (extract from Footer.astro lines 52-104):
   - "How long does a typical project take?"
   - "Do you work with clients outside your area?"
   - "What if I'm not sure exactly what I need?"
   - "How do you handle changes during the project?"
   - "What happens after handover?"

3. Create FAQPage JSON-LD schema following Google's specification:
```javascript
const faqSchema = {
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": faqs.map(faq => ({
    "@type": "Question",
    "name": faq.question,
    "acceptedAnswer": {
      "@type": "Answer",
      "text": faq.answer
    }
  }))
};
```

4. Page layout:
   - Use BaseLayout with title="FAQ" description="Frequently asked questions about working with Joel Shinness"
   - Add JSON-LD script in head slot: `<script type="application/ld+json" set:html={JSON.stringify(faqSchema)} />`
   - Main content: h1 "Frequently Asked Questions", then accordion using native `<details>` elements
   - Match Footer.astro accordion styling: border-[3px], rounded, p-4, group class for icon animation
   - Use same color tokens: text-text-light, dark:text-text-dark, etc.
   - Container: max-w-3xl mx-auto for readability
   - Section padding: py-16 px-4 (consistent with other sections)

5. Map over faqs array to render each as:
```html
<details class="group border-[3px] border-text-light dark:border-text-dark rounded p-4">
  <summary class="font-heading font-bold cursor-pointer list-none flex items-center justify-between text-text-light dark:text-text-dark">
    <span>{question}</span>
    <span class="text-2xl transition-transform group-open:rotate-45">+</span>
  </summary>
  <div class="mt-4 pt-4 border-t-[3px] border-text-light/20 dark:border-text-dark/20 text-text-muted-light dark:text-text-muted-dark leading-relaxed">
    {answer}
  </div>
</details>
```

Native `<details>` elements provide built-in keyboard accessibility (Tab navigation, Enter/Space toggle) - no JavaScript required.
