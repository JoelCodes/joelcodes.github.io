     - Yellow (default) - primary CTA
     - Turquoise - secondary actions
     - Magenta - accent/special actions
   - Live examples: `<Button variant="yellow">Yellow</Button>` etc.

4. **Size variants:**
   - sm: `px-4 py-2 text-sm`
   - md: `px-6 py-3 text-base` (default)
   - lg: `px-8 py-4 text-lg`
   - Live examples of each size

5. **Link variant:**
   - Using `href` prop: `<Button href="/contact">Contact</Button>`
   - Renders as `<a>` instead of `<button>`

6. **States documentation:**
   - Hover: lifts up, shadow grows
   - Active: presses down, shadow shrinks
   - Focus: double ring (WCAG 2.4.13 compliant)
   - Disabled: reduced opacity (document via prose)

   ```astro
   <Button variant="yellow">Primary Action</Button>

   ```

8. **Props table:**
   | Prop | Type | Default | Options |

   | size | string | 'md' | 'sm', 'md', 'lg' |
